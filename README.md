## Installation
- rename secrets.yml.example to secrets.yml and put your data in it  
- run conky_start

### Dependencies
Conky libs:  
  - conky-imlib2

Gmail:  
  - ruby >= 1.9.3

  gems:  
  - ruby-gmail
  - mime
  - rest-client

Weather:  
  - api_key from http://www.wunderground.com/weather/api/
